Data Sources
The data for this analysis was sourced from the following sources:
Zillow: Median House Prices (USD) based on Zillow Home Value Index (ZHVI) for single family residences (1-4 bedrooms) (1996-2020)
Census Data: Population, Poverty Rate, and Median Household Income (USD) (2018)
HIFLD Schools Data: All Public, K-12 schools data (2017-2018)
Data Preprocessing & ETL
Loaded the data from all sources into Pandas DataFrames
Filtered, grouped, renamed and reordered attributes to prepare for aggregations using Pandas
Deleted null values from predictors
Converted date values to seconds to prepare for machine learning
Merged the transformed data to use in the machine learning algorithms
Converted the house prices to logarthmic values to ensure symmetrical distribution
Converted the categorical predictors (City, Zipcode) into one-hot vectors
Exploratory & Predictive Analysis using Tableau Public & Python Libraries (Pandas, Scikit-learn, Matplotlib, Seaborn, Numpy)
Identified trends in house prices across North Carolina using Tableau Public
Normalized the house prices to use in machine learning algorithms
Identified correlations between various features effecting house prices using Numpy, Matplotlib & Seaborn libraries
Created multiple, static and dynamic visualizations using Tableau Public
Predicted house pricies with Linear Regression using Scikit-learn and Matplotlib
