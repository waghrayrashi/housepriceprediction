Data Sources
The data for this analysis was sourced from the following sources:
Zillow: Median House prices ($) based on Zillow Home Value Index (ZHVI) for single family residences (1-4 bedrooms)
Census Data: Population, Poverty Rate, and Median Household Income ($)
HIFLD Schools Data: Feature dataset of all Public, K-12 schools (2017-2018)
ETL Processes Employed for Anlaysis
Filtered, grouped and performed aggregations on relevant attributes using Pandas
Changed datatypes of relevant attributes
Deleted null values from analysis
Exploratory & Predictive Analysis using Tableau Public & Python Libraries (Pandas, Scikit-learn, Matplotlib, Numpy)
Identified trends in house prices across North Carolina
Identified correlations between various features effecting house prices
Explored the relationship between income, poverty rate, number of public schools and Median House Prices
Predicted house pricies with Linear Regression using Scikit-learn and matplotlib
Created multiple, static and dynamic visualizations using Tableau Public
